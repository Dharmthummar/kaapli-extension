Easy Installation

Get started with kaapli in just a few simple steps.

Download the Extension

Download the latest version from our GitHub repository.

Download Latest Version

Extract the Files

Unzip the downloaded file to a folder on your computer where you want to store the extension files.

Open Chrome Extensions

Navigate to chrome://extensions/ in your Chrome browser.

Enable Developer Mode

Toggle on Developer Mode in the top-right corner of the extensions page.

Load the Extension

Click on "Load unpacked" and select the folder where you extracted the extension files.

Set Up Your API Key

Click on the extension icon, enter your Gemini API key, and start using kaapli!

Frequently Asked Questions

Find answers to common questions about kaapli.

How do I get a Gemini API key?

You can get a free Gemini API key by visiting the Google AI Studio. Sign in with your Google account, navigate to the API section, and create a new API key. It's completely free to use within the limits.

Can I use kaapli with other websites?

Yes, kaapli is designed to work with various websites. However, it may not work with all websites due to the nature of the extension. You can try it out on a few websites to see if it works for you.

Is kaapli safe to use?

Yes, kaapli is designed to be safe and secure. It uses the Gemini API, which is a secure and reliable service provided by Google. Your API key is kept secure and is not shared with anyone.
